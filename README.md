# audit-record

#### 介绍
##### 一个流式风格的记录器，它提供了如下能力：
- 支持spel表达式和注解的方式进行方法调用记录
- 支持方法路由的方式进行方法调用记录
- 支持全局和局部记录数的统一处理
- 支持同步和异步的方法调用记录 
- 对spring boot进行了无缝的集成 
- 支持用户自定义扩展
